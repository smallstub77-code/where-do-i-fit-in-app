Methodological Notes

Important context for interpreting the report.

Measurement method

Length values are intended to be measured along the upper surface using the bone-pressed method.

Self-reported values

The report relies on measurements submitted by the participant and cannot independently verify measurement technique.

Percentile estimates

Percentiles describe position within the selected reference model and should be interpreted as estimates rather than diagnoses.

Missing measurements

Only submitted values are analysed. Missing entries remain unestimated and are not treated as zero.

No products to sell. This project does not promote supplements, medications, enlargement products, performance enhancers, coaching programmes, or commercial treatments based on report results.
